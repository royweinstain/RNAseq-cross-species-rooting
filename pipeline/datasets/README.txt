Datasets Directory
==================

Each subdirectory represents one dataset (species + genotype comparison).

Structure:
  datasets/<species>_<easyGenotype>_vs_<hardGenotype>/
    raw/           Raw expression file (Excel or CSV)
    processed/     DE results generated by Step 1 (de_results.csv)

Adding a New Dataset:
  1. Create folder: datasets/<species>_<easy>_vs_<hard>/raw/
  2. Place the raw expression file in raw/
  3. Add a species entry in config.yaml with:
       - name: species name (matches orthofinder_column species)
       - dataset_id: folder name (must be unique)
       - expression_file, expression_sheet, header_row, gene_id_column
       - gene_id_prefixes, orthofinder_column, data_format
       - samples: easy: [...] and hard: [...]
       - de_output: datasets/<folder>/processed/de_results.csv
  4. Run: python run_pipeline.py config.yaml --steps 1 --dataset <dataset_id>
  5. Run: python run_pipeline.py config.yaml --steps 2,3,4,5

Naming Convention:
  <species>_<easyGenotype>_vs_<hardGenotype>
  Examples: argan_196_vs_YM3, poplar_OP42_vs_T89, poplar_G_vs_B
